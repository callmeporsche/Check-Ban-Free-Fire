# 📝 API Documentation

# 🪪 CHECK IF ANY ACCOUNT IS BANNED

```http
GET http://localhost/isbanned.php?uid={uid}
GET http://yourdomain/isbanned.php?uid={uid}
```


### 💬 Response Example
```json
{
  "message": "Account Has Been Permanently Banned💀"
}
{
  "message": "Account Not Banned"
}
```
